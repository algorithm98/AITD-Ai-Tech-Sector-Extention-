🧠 AI Tech Detector (AITD)

Detect AI tools and technologies used in any website — instantly.

⸻

🚀 Overview

AI Tech Detector (AITD) is a powerful browser extension that scans websites to identify AI technologies, APIs, and frameworks used in their development.

Built for developers, marketers, and agencies, AITD helps you:

* 🔍 Analyze competitor tech stacks
* 🧠 Discover AI tools powering modern apps
* ⚡ Gain insights into AI-driven products

⸻

✨ Features

* 🔎 AI Detection Engine
    Detects popular AI technologies such as:
    * OpenAI (GPT-based systems)
    * Anthropic (Claude)
    * Google AI (Gemini)
    * Hugging Face
* 🌐 Network Request Monitoring
    Identifies hidden API calls to AI services
* 🧠 Heuristic-Based Detection
    Uses DOM scanning, script analysis, and keyword matching
* 🎨 Modern UI
    Clean, animated popup interface with real-time results
* ⚡ Lightweight & Fast
    Runs instantly on any webpage

⸻

📦 Installation

🔧 Load Unpacked Extension

1. Clone the repository:

git clone https://github.com/algorithm98/AITD-Ai-Tech-Sector-Extention-.git

2. Open Chrome and go to:

chrome://extensions/

3. Enable Developer Mode
4. Click Load Unpacked
5. Select the project folder

⸻

🧪 Usage

1. Open any website
2. Click the AI Tech Detector extension
3. Click “Scan Website”
4. View detected AI technologies with confidence scores

⸻

🧠 How It Works

AITD uses a combination of:

* 🔍 DOM Analysis → Detects keywords & UI patterns
* 📜 Script Inspection → Identifies AI libraries
* 🌐 Network Monitoring → Tracks API calls to AI providers

This creates a probability-based detection system for identifying AI usage.

⸻

⚠️ Limitations

* Cannot fully detect server-side AI implementations
* Proxy APIs may hide real providers
* Results are heuristic-based (not guaranteed 100%)
